# Lets start-

**The site is responsive and suitable for viewing on a computer screen, tablet or phone.**<br>

In order to run this project on VSC , you will need to run those terminal commands-
```
npm install
```
## On PopWiz-client

```
npm start
```

## On PopWiz-server
```
nodemon
```
